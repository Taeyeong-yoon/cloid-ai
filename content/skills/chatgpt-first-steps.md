## 🎯 학습 목표

1. ChatGPT 계정을 직접 만들고 로그인할 수 있다
2. 대화창에서 질문을 입력하고 답변을 받을 수 있다
3. 대화를 이어가며 원하는 결과를 얻는 기본 흐름을 익힌다

---

## 📖 ChatGPT란?

ChatGPT는 OpenAI가 만든 AI 대화 도구입니다. 마치 매우 똑똑한 비서에게 질문하듯 텍스트로 대화하면, AI가 답변·글쓰기·번역·코딩 등 다양한 작업을 도와줍니다.

---

## 🛠️ 실습 1 — 회원가입 & 첫 로그인

**단계별 가이드:**

1. 브라우저에서 `chat.openai.com` 접속
2. **[Sign up]** 버튼 클릭
3. 이메일 또는 구글/애플 계정으로 가입
4. 이메일 인증 링크 클릭
5. 이름과 생년월일 입력 후 완료
6. 무료 플랜(Free)으로 시작 — 신용카드 불필요

**로그인 후 화면 구성:**
- 왼쪽: 대화 기록 목록
- 가운데 하단: 메시지 입력창
- 오른쪽 상단: 모델 선택 (GPT-5.2 권장)

---

## 🛠️ 실습 2 — 첫 대화 보내기

아래 프롬프트를 입력창에 그대로 붙여넣고 Enter를 눌러보세요.

```
안녕하세요! 저는 ChatGPT를 처음 사용해보는 초보자예요.
ChatGPT가 어떤 일을 도와줄 수 있는지 5가지만 쉽게 설명해주세요.
```

답변이 나오면 바로 이어서:

```
방금 말한 5가지 중에서 직장인이 가장 많이 쓰는 것은 무엇인가요?
이유도 함께 알려주세요.
```

이처럼 **대화를 이어가는 것**이 ChatGPT 활용의 핵심입니다.

---

## 💬 프롬프트 템플릿 5개

### 1. 개념 설명 요청
```
[주제]를 전혀 모르는 초보자에게 설명하듯이 쉽게 알려주세요.
예시도 2개 포함해서요.
```

### 2. 할 일 목록 정리
```
오늘 해야 할 일이 있어요:
[할 일들을 나열]

이것들을 우선순위 순서로 정리하고,
각각 예상 소요 시간도 알려주세요.
```

### 3. 이메일 초안 작성
```
아래 상황에 맞는 이메일을 써주세요.
- 받는 사람: [대상]
- 목적: [목적]
- 톤: 정중하고 간결하게
- 길이: 3~4문장
```

### 4. 모르는 단어 설명
```
"[단어 또는 용어]"가 무슨 뜻인가요?
일상적인 예시를 들어 설명해주세요.
```

### 5. 아이디어 브레인스토밍
```
[주제]에 관한 아이디어를 10개 제안해주세요.
각 아이디어는 한 줄로 간결하게 써주세요.
```

## 🎯 도전 과제

**미션**: ChatGPT와 5번 이상 대화를 주고받아 원하는 결과물 하나를 완성하세요.

**완성 조건:**
- 첫 질문 → AI 답변 → 수정 요청 → 최종 결과물 흐름 완성
- 결과물 예시: 자기소개서 한 단락, 업무 이메일, 오늘의 식단 계획 중 택 1

---

## ⚠️ 자주 하는 실수 3가지

**1. 너무 짧고 모호한 질문**
- ❌ "이메일 써줘"
- ✅ "거래처에 납품 지연을 사과하는 정중한 이메일을 3문장으로 써줘"
- 💡 **해결법**: 대상·목적·형식·길이를 함께 명시하세요

**2. 한 번에 너무 많은 요청**
- ❌ "번역하고, 요약하고, 이메일로 만들어줘"
- ✅ 번역 → 요약 → 이메일 순으로 단계별 요청
- 💡 **해결법**: 하나씩 차례로 대화하세요

**3. 첫 답변에 만족하고 끝내기**
- ❌ 첫 답변을 그대로 사용
- ✅ "더 짧게", "좀 더 격식체로", "3번째 항목을 더 자세히" 등으로 수정 요청
- 💡 **해결법**: AI는 무한 수정이 가능합니다. 원하는 결과가 나올 때까지 대화하세요
