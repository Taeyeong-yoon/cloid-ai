1. 블로그·SNS·이메일 각 용도에 맞는 AI 글쓰기 프롬프트를 작성할 수 있다

## 📖 AI 글쓰기의 핵심 원칙

AI는 **초안 생성기**입니다. 10분 걸리던 글이 1분 초안으로 줄고, 나머지 9분을 편집과 개성 추가에 쓸 수 있습니다.

**AI 글쓰기 3단계 흐름:**
1. **구조 잡기** → 아이디어·목차 생성 요청
2. **초안 작성** → 본문 생성 요청
3. **퇴고** → 톤 조정, 길이 조정, 내 표현으로 수정

## 🛠️ 실습 1 — 블로그 포스트 처음부터 끝까지

**1단계: 목차 생성**
```
"재택근무 3년 차가 깨달은 집중력 유지 비법"이라는 블로그 주제로
독자가 끝까지 읽고 싶은 목차를 5개 섹션으로 만들어주세요.
각 섹션에 소제목과 다룰 내용을 한 줄씩 적어주세요.
```

**2단계: 도입부 작성**
```
위 목차를 바탕으로 블로그 도입부를 써주세요.
- 독자의 공감을 끌어내는 문제 제시로 시작
- "나도 그랬다"는 개인 경험 1줄 포함
- 이 글을 읽으면 얻을 것을 마지막 문장에 넣기
- 분량: 150자 내외
```

## 🛠️ 실습 2 — SNS 콘텐츠 빠르게 만들기

하나의 아이디어로 여러 플랫폼용 콘텐츠를 동시에 만드세요.

```
아이디어: "AI를 쓰면 업무 시간이 절반으로 줄어든다"

이 아이디어로 아래 3가지를 각각 만들어주세요:
1. 인스타그램 캡션 (3문장 + 해시태그 5개)
2. 링크드인 포스트 (전문적인 톤, 5~7문장)
3. 트위터/X 스레드 (3개 연속 트윗, 각 140자 이내)
```

## 💬 프롬프트 템플릿 (용도별)

### 블로그

#### 1. 블로그 글 전체 작성
```
주제: "[주제]"
대상 독자: [독자층]
목적: [정보 제공/공감 유발/구매 유도]
분량: [800/1500/2000자]
톤: [친근한/전문적/감성적]
구조: 도입 → 본론 3개 항목 → 결론 + CTA

위 조건으로 블로그 글을 작성해주세요.
```

#### 2. 블로그 제목 10개 생성
```
"[주제]"에 관한 블로그 포스트 제목을 10개 만들어주세요.
조건: 클릭하고 싶은 제목, 숫자 포함, 감정을 자극하는 표현 사용
```

### SNS

#### 3. 인스타그램 캡션
```
[사진/상황 설명]에 어울리는 인스타그램 캡션을 써주세요.
분위기: [밝은/감성적/유머러스]
길이: 3~5문장
해시태그: 관련 해시태그 7개 포함
```

#### 4. 링크드인 포스트
```
아래 경험/인사이트를 링크드인 포스트로 작성해주세요.
내용: [공유할 내용]
목적: [네트워킹/브랜딩/취업 어필]
톤: 전문적이지만 인간적인 느낌
길이: 200~300자
끝에 질문으로 마무리해주세요.
```

#### 5. 유튜브 영상 설명란
```
아래 영상 내용의 유튜브 설명란을 작성해주세요.
영상 주제: [주제]
주요 내용: [내용 요약]
포함 사항: 타임스탬프 3개, 관련 링크 공간, 채널 구독 유도
SEO용 키워드: [키워드 3~5개]
```

### 이메일/문서

#### 6. 뉴스레터 이메일
```
아래 주제로 구독자에게 보내는 뉴스레터를 작성해주세요.
주제: [이번 주 뉴스레터 주제]
구독자 특성: [직업/관심사]
구성: 인사 → 핵심 내용 3가지 → 마무리 + CTA
분량: 400자 내외
친근하고 직접 말하는 듯한 톤
```

#### 7. 제안서 도입부
```
[서비스/제품명] 제안서의 도입부를 작성해주세요.
클라이언트 문제: [문제 상황]
우리의 해결책: [솔루션 요약]
목적: 첫 문단에서 클라이언트가 계속 읽고 싶게 만들기
분량: 150자 이내
```

## 🎯 도전 과제

**미션**: 내 관심 주제로 블로그 포스트 1개를 AI와 함께 완성하세요.

**완성 조건:**
- 목차 → 초안 → 퇴고 3단계 프로세스 완료
- 최종 글에서 AI가 쓴 부분과 내가 수정한 부분을 색깔로 구분 표시
- 완성된 글 800자 이상

## ⚠️ 자주 하는 실수 3가지

**1. AI 말투가 그대로 남아있음**
- ❌ "물론입니다!", "확실히", "중요합니다" 같은 AI 특유 표현
- ✅ 내 말투로 바꾸는 퇴고 단계를 반드시 거치기
- 💡 **해결법**: "AI 티가 나는 표현을 제거하고 자연스러운 한국어로 다듬어줘"로 재요청

**2. 너무 형식적인 글만 나옴**
- ❌ 딱딱하고 교과서 같은 문체
- ✅ "친구에게 카톡 보내는 말투로", "유머를 살짝 섞어서" 등 톤을 구체적으로 지정
- 💡 **해결법**: 내가 좋아하는 블로거/작가 스타일을 참고해서 "이런 스타일로" 요청

**3. 사실 확인 없이 AI 내용을 그대로 게시**
- ❌ AI가 생성한 통계, 사실, 링크를 검증 없이 사용
- ✅ 수치·사실·인용 등은 반드시 원본 확인 후 사용
- 💡 **해결법**: AI에게 "출처를 알려줘"라고 물어보고, 실제 검색으로 확인
