## 🎯 학습 목표

1. AI에게 웹앱 코드를 요청하고 실제로 실행하는 방법을 익힌다
2. 에러가 발생했을 때 AI와 함께 디버깅하는 흐름을 경험한다
3. 간단한 기능 추가·수정을 AI 대화로 반복하는 개발 사이클을 완성한다

---

## 📖 비개발자의 AI 코딩 전략

개발자처럼 코드를 쓸 필요 없습니다. 대신 아래 3가지 능력을 키우세요:

1. **원하는 것을 정확히 설명하는 능력** — 기능을 글로 설명
2. **코드를 실행하는 방법** — 어디에 붙여넣어야 하는지
3. **오류를 AI에게 전달하는 방법** — 에러 메시지 그대로 전달

---

## 🛠️ 실습 1 — HTML 한 파일로 ToDo 앱 만들기

아래 프롬프트를 Claude 또는 ChatGPT에 입력하세요.

```
HTML, CSS, JavaScript를 사용해서 ToDo 앱을 만들어주세요.

요구사항:
- 파일 하나 (index.html)로 완성
- 할 일 추가 (입력창 + 버튼)
- 할 일 완료 체크 (체크박스 클릭 시 취소선)
- 할 일 삭제 버튼
- 새로고침해도 목록 유지 (localStorage 사용)
- 디자인: 깔끔하고 모던하게, 보라색 계열 색상

초보자도 이해할 수 있도록 코드에 한국어 주석 달아주세요.
```

받은 코드를 메모장에 붙여넣고 `todo.html`로 저장 후 더블클릭해서 실행하세요.

## 🛠️ 실습 2 — 기능 추가와 디버깅

실습 1에서 만든 앱에 기능을 추가해보세요.

```
위에서 만든 ToDo 앱에 아래 기능을 추가해주세요:

추가 기능:
1. 할 일에 우선순위 설정 (높음/보통/낮음) + 색상 구분
2. 완료된 항목 일괄 삭제 버튼
3. 전체/미완료/완료 필터 탭

기존 코드 전체를 수정된 버전으로 다시 출력해주세요.
```

## 💬 프롬프트 템플릿 8개

### 1. 간단한 웹 계산기
```
HTML+CSS+JS로 계산기를 만들어주세요.
- 사칙연산 (+, -, ×, ÷)
- 소수점 지원
- 연속 계산 가능
- 디자인: [원하는 스타일]
- 하나의 HTML 파일로 완성
```

### 2. 타이머 / 스톱워치
```
웹 타이머를 만들어주세요.
기능:
- 카운트다운 타이머 (시간 설정 가능)
- 스톱워치 (시작/일시정지/리셋)
- 타이머 종료 시 알림음
HTML+CSS+JS 단일 파일로 만들어주세요.
```

### 3. 간단한 폼 + 데이터 저장
```
설문조사 폼을 만들어주세요.
질문 목록: [질문들]
저장 방식: localStorage에 저장
결과 보기: 제출 후 결과 요약 화면 표시
디자인: 깔끔하고 모바일 반응형
```

### 4. API 연동 날씨 앱
```
OpenWeatherMap API를 사용하는 날씨 앱을 만들어주세요.
- 도시 이름 검색
- 현재 날씨 (온도, 습도, 날씨 상태) 표시
- 날씨 아이콘 표시
- API 키를 코드에 넣는 위치 주석으로 표시
HTML+CSS+JS 단일 파일
```

### 5. 에러 디버깅 요청
```
아래 코드를 실행했더니 에러가 납니다.

에러 메시지: [에러 복사 붙여넣기]
코드: [문제 코드]
브라우저: [크롬/파이어폭스 등]

원인을 찾아 수정한 전체 코드를 주세요.
```

### 6. 기존 코드에 기능 추가
```
아래 기존 코드에 [기능]을 추가해주세요.

기존 코드:
[코드 붙여넣기]

추가할 기능:
- [기능 1]
- [기능 2]

수정된 전체 코드를 다시 출력해주세요.
```

### 7. 코드 설명 요청
```
아래 코드가 어떻게 동작하는지 설명해주세요.
코딩을 모르는 사람도 이해할 수 있는 수준으로,
각 중요한 부분을 한 줄씩 설명해주세요.

[코드 붙여넣기]
```

## 🎯 도전 과제

**미션**: 나의 실제 필요를 해결하는 웹 도구 하나를 AI와 함께 만드세요.

**완성 조건:**
- 아이디어 설명 → AI 코드 생성 → 실행 → 수정 요청 → 최종 완성 흐름 완료
- 실제로 브라우저에서 동작하는 HTML 파일 1개
- 최소 기능 3가지 이상 포함

---

## ⚠️ 자주 하는 실수 3가지

**1. 기능을 한 번에 너무 많이 요청**
- ❌ 20가지 기능을 한 번에 요청 → 코드가 너무 복잡해져 오류 발생
- ✅ 핵심 기능 3가지로 시작 → 동작 확인 후 기능 추가
- 💡 **해결법**: MVP(최소 기능 제품) 먼저, 기능 추가는 단계별로

**2. 에러 발생 시 처음부터 다시 요청**
- ❌ 오류 나면 새 대화 열고 처음부터 재요청
- ✅ 같은 대화에서 에러 메시지를 그대로 복사해 "이 에러 고쳐줘"
- 💡 **해결법**: 브라우저 개발자 도구(F12) → Console 탭의 빨간 에러 메시지 복사

**3. 코드를 이해하려 하지 않음**
- ❌ AI가 만든 코드를 블랙박스로 취급
- ✅ "이 부분이 무슨 역할인지 설명해줘"로 핵심 부분 이해
- 💡 **해결법**: 전체를 다 몰라도 OK. 각 함수가 뭘 하는지 정도는 파악하세요
