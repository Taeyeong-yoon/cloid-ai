## 무엇인가?

Artifacts는 Claude가 생성한 결과물을 대화창 옆 별도 패널에 표시하는 기능입니다. 코드를 바로 실행하거나, HTML을 렌더링하거나, 문서를 편집할 수 있습니다.

## 지원하는 Artifact 유형

| 유형 | 설명 | 예시 |
|------|------|------|
| **Code** | 코드 하이라이팅 + 복사 | Python, JS, SQL |
| **HTML** | 브라우저 미리보기 | 웹페이지, 대시보드 |
| **React** | 인터랙티브 컴포넌트 실행 | UI 프로토타입 |
| **SVG** | 벡터 이미지 렌더링 | 다이어그램, 아이콘 |
| **Markdown** | 서식 있는 문서 | 보고서, 문서 |
| **Mermaid** | 플로우차트·시퀀스 다이어그램 | 아키텍처 설계도 |

## 활용법

### HTML 대시보드 즉시 생성
```
판매 데이터를 시각화하는 HTML 대시보드를 만들어줘.
차트는 Chart.js 사용, 다크 테마로.
```
→ Artifact에서 바로 렌더링 확인 후 다운로드

### React 프로토타입
```
투두리스트 React 컴포넌트 만들어줘.
추가/삭제/완료 체크 기능 포함.
```
→ Artifact에서 실제로 클릭해서 테스트 가능

### 반복 수정
Artifact가 열린 상태에서 "배경색을 파란색으로", "폰트 크기 키워줘" 등 자연어로 수정 요청하면 실시간 업데이트됩니다.

## 팁

- Artifact 우측 상단 **Share** 버튼으로 링크 공유 가능
- 여러 Artifact를 만들면 대화 내에서 히스토리처럼 관리됨
- Claude.ai 모바일 앱에서도 Artifact 확인 가능
