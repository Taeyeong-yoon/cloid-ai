## 무엇인가?

Custom Instructions는 claude.ai 전체 대화에 적용되는 나만의 기본 설정입니다. "나는 개발자야", "한국어로 답해줘", "이모지 쓰지 마" 같은 조건을 한 번만 저장하면 됩니다.

## 설정 위치

claude.ai → 좌측 하단 프로필 아이콘 → **Settings** → **Profile**

## 작성 예시 — 개발자

```
나는 5년차 풀스택 개발자입니다.
주 언어는 TypeScript + Python입니다.

응답 스타일:
- 핵심 먼저, 설명은 간결하게
- 코드 예시는 항상 TypeScript 기준으로
- 당연한 설명(변수 선언이 뭔지 등)은 생략
- 마크다운으로 구조화해서 작성
- 한국어로 답변
```

## 작성 예시 — 학생·연구자

```
나는 대학원에서 AI를 공부하는 학생입니다.
주제: 자연어처리, LLM 파인튜닝

응답 스타일:
- 수식과 논문 참고문헌 적극 활용
- 쉬운 비유보다 정확한 용어 선호
- 핵심 개념 설명 후 실제 구현 예시 포함
- 영어 원문 용어는 괄호로 병기
```

## 작성 예시 — 소상공인

```
저는 카페를 운영하는 자영업자입니다.
IT 지식은 많지 않습니다.

응답 스타일:
- 전문 용어 없이 쉽게 설명
- 실제로 따라할 수 있는 단계별 안내
- 짧고 명확하게, 긴 설명은 피해주세요
- 한국어로만 답변
```

## 팁

- Custom Instructions는 Projects 설정보다 우선순위가 낮습니다
- 지시사항이 충돌하면 Project Instructions가 우선 적용됩니다
- 300~500자 이내로 핵심만 작성하는 것이 효과적입니다
- 정기적으로 업데이트하면 점점 더 맞춤화된 경험을 할 수 있습니다
