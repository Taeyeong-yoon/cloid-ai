## 지원 파일 형식

| 종류 | 형식 |
|------|------|
| **문서** | PDF, TXT, MD, DOCX |
| **이미지** | PNG, JPG, GIF, WebP |
| **코드** | 모든 텍스트 기반 코드 파일 |
| **데이터** | CSV (텍스트로 처리) |
| **스프레드시트** | XLSX (내용 추출) |

## 활용 시나리오

### 계약서·보고서 분석
```
[계약서.pdf 첨부]
이 계약서에서 해지 조건과 위약금 관련 조항을 찾아서 요약해줘.
불리한 조항이 있으면 표시해줘.
```

### 이미지 속 텍스트 추출 (OCR)
```
[영수증 사진 첨부]
이 영수증의 항목별 금액을 표로 정리해줘.
```

### 코드베이스 이해
```
[legacy_code.py 첨부]
이 코드가 무엇을 하는지 설명해주고,
개선할 수 있는 부분을 3가지 찾아줘.
```

### 데이터 분석
```
[sales_data.csv 첨부]
이 데이터에서 월별 매출 트렌드를 분석하고,
가장 실적이 좋은 제품 카테고리 Top 3를 알려줘.
```

### 화면 캡처로 버그 리포트
```
[에러 스크린샷 첨부]
이 에러 메시지의 원인이 뭔지 알려주고 해결 방법 알려줘.
```

## 용량 제한

- 파일당 최대 **32MB**
- 이미지는 최대 **5장**까지 한 번에 업로드
- PDF는 **최대 100페이지** 권장 (그 이상은 잘릴 수 있음)

## 팁

- 여러 파일을 한 번에 올리고 "이 파일들을 비교해줘"도 가능
- 이미지 품질이 낮으면 텍스트 인식 정확도가 떨어집니다
- 민감한 개인정보가 포함된 파일 업로드 시 주의
