## 🎯 학습 목표

1. 단순 번역과 맥락 기반 번역의 차이를 이해하고 적용할 수 있다
2. 문서 종류와 상대방에 맞는 톤으로 번역을 요청할 수 있다
3. 번역을 넘어 현지화(Localization)까지 AI로 처리할 수 있다

---

## 📖 AI 번역이 구글 번역과 다른 점

구글 번역: 단어·문장을 기계적으로 변환
AI 번역: 맥락·톤·문화적 뉘앙스를 이해하고 자연스럽게 변환

**AI 번역의 5가지 강점:**
- 비즈니스/캐주얼/학술 등 톤 지정 가능
- 용어 일관성 유지 지시 가능
- 현지 문화에 맞는 표현으로 현지화
- 번역 이유 설명 요청 가능
- 원문 의도 보존 여부 확인 가능

---

## 🛠️ 실습 1 — 비즈니스 이메일 번역

```
아래 한국어 이메일을 비즈니스 영어로 번역해주세요.

조건:
- 받는 사람: 미국 파트너사 임원
- 관계: 처음 연락하는 사이
- 톤: 격식체, 정중하고 자신감 있게
- 주의: 직역하지 말고 영어 이메일 문화에 맞게 자연스럽게

[번역할 이메일 내용]
```

## 🛠️ 실습 2 — 번역 품질 비교 실험

같은 문장을 3가지 방법으로 번역해보세요.

```
아래 문장을 3가지 방법으로 번역해주세요:
"이 제품은 특별한 날을 더욱 특별하게 만들어 드립니다."

1. 직역 (word-for-word)
2. 자연스러운 영어 표현으로
3. 미국 광고 카피 스타일로

각 번역의 특징과 어떤 상황에 쓰면 좋은지 설명해주세요.
```

## 💬 프롬프트 템플릿 5개

### 1. 맥락 기반 번역
```
아래 내용을 [목표 언어]로 번역해주세요.

맥락 정보:
- 문서 종류: [계약서/이메일/SNS/광고 등]
- 대상 독자: [국가, 연령대, 직업]
- 톤: [격식체/캐주얼/친근한/전문적]
- 주의사항: [회사명, 고유명사, 특정 용어는 번역하지 말 것]

[번역할 내용]
```

### 2. 역번역 검증
```
아래 번역이 원문의 의도를 정확히 전달하는지 확인해주세요.

원문(한국어): [원문]
번역본(영어): [번역본]

확인 요청 사항:
1. 의미가 정확히 전달되었는지
2. 어색한 표현이 있는지
3. 더 나은 대안 표현이 있는지
```

### 3. 현지화 번역
```
아래 한국어 마케팅 문구를 미국 시장용으로 현지화해주세요.

단순 번역이 아니라:
- 미국 소비자에게 친숙한 표현 사용
- 문화적으로 어색한 부분 수정
- 같은 감정과 행동을 유발하는 미국식 표현으로

원문: [내용]
```

### 4. 다국어 동시 번역
```
아래 공지사항을 영어, 일본어, 중국어(간체)로 각각 번역해주세요.
각 언어의 문화에 맞는 자연스러운 표현을 사용해주세요.
회사명 "[회사명]"은 번역하지 마세요.

[번역할 내용]
```

### 5. 번역 용어집 생성
```
아래 분야의 전문 용어를 한국어-영어 대응 번역 용어집으로 만들어주세요.

분야: [IT/의학/법률/금융 등]
용어 수: 20개
형식: 표 형태 (한국어 | 영어 | 사용 예시)
```

## 🎯 도전 과제

**미션**: 실제 업무에서 쓸 문서 하나를 AI로 번역하고 원어민 수준의 결과물을 완성하세요.

**완성 조건:**
- 번역 대상 문서 선정 (이메일, 보고서, 제안서 중 택 1)
- 초안 번역 → 톤·맥락 수정 요청 → 최종본 완성
- 구글 번역 결과와 AI 번역 결과 비교 스크린샷 저장

## ⚠️ 자주 하는 실수 3가지

**1. 맥락 없이 번역만 요청**
- ❌ "영어로 번역해줘" + 문장 붙여넣기
- ✅ 문서 종류, 대상 독자, 톤을 항상 함께 명시
- 💡 **해결법**: 번역 전 "이 문서의 용도는 ~, 읽는 사람은 ~"을 한 줄 추가

**2. 고유명사/브랜드명까지 번역됨**
- ❌ 회사명, 제품명, 사람 이름이 임의로 번역됨
- ✅ "다음 단어는 번역하지 마세요: [목록]" 명시
- 💡 **해결법**: 번역 금지 목록을 프롬프트 맨 앞에 미리 제시

**3. AI 번역을 검토 없이 최종 사용**
- ❌ 중요한 계약서나 공식 문서를 AI 번역 그대로 제출
- ✅ 중요 문서는 원어민 또는 전문 번역가의 검토 필수
- 💡 **해결법**: AI 번역은 초안 작성용, 최종 검토는 사람이 담당
