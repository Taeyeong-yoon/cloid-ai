## 🛠️ 실습 1 — DALL-E로 첫 이미지 생성

ChatGPT에 접속해서 아래 프롬프트를 입력해보세요.

```
아래 내용으로 이미지를 생성해주세요:
- 장면: 현대적인 카페에 앉아 노트북으로 일하는 30대 여성
- 스타일: 따뜻한 색감의 사진 같은 사실적 표현
- 분위기: 아늑하고 생산적인 느낌
- 조명: 창문으로 들어오는 오후 햇살
```

## 💬 프롬프트 템플릿 5개

### 1. 사실적 인물 사진
```
[나이]대 [성별], [직업/상황]을 하고 있는 모습.
배경: [장소]
조명: [자연광/스튜디오/역광 등]
스타일: 사실적인 사진 (photorealistic)
분위기: [키워드]
```

### 2. 제품 광고 이미지
```
[제품명]의 제품 광고 사진.
배경: 깔끔한 흰색 또는 [색상] 배경
조명: 스튜디오 조명, 그림자 없음
스타일: 미니멀하고 고급스러운 상업 사진
[제품]을 중앙에 배치
```

### 3. 일러스트 스타일
```
[주제]를 [스타일] 일러스트로 그려주세요.
색상 팔레트: [파스텔/모노크롬/비비드 등]
선 스타일: [굵은 선/섬세한 선/선 없음]
분위기: [귀여운/모던/레트로 등]
```

### 4. 로고/아이콘 디자인
```
[브랜드명]을 위한 [업종] 로고 디자인.
스타일: 미니멀, 벡터 아트
색상: [주요 색상] 계열
심볼: [원하는 이미지/개념] 연상
배경: 투명
```

### 5. SNS 썸네일
```
유튜브 썸네일 이미지를 만들어주세요.
주제: "[영상 제목]"
텍스트: "[썸네일에 넣을 문구]" (눈에 띄는 크고 굵은 폰트)
배경: [색상 또는 장면]
표정/분위기: 놀람/기대감을 주는
```

## 🎯 도전 과제

**미션**: 세 가지 도구(DALL-E, Ideogram, 그리고 하나 더)에서 동일한 주제로 이미지를 각각 생성하고 비교하세요.

**완성 조건:**
- 동일한 주제로 3가지 도구에서 각 1장씩 생성
- 각 도구의 결과물에서 마음에 드는 점 / 아쉬운 점을 한 줄씩 기록
- 어떤 용도에 어떤 도구가 맞는지 나만의 결론 작성

---

## ⚠️ 자주 하는 실수 3가지

**1. 프롬프트가 너무 짧음**
- ❌ "카페에서 일하는 사람"
- ✅ 주제 + 스타일 + 조명 + 분위기 + 기술 사양을 구체적으로 명시
- 💡 **해결법**: 영어로 작성하면 대부분 도구에서 더 좋은 결과가 나옵니다

**2. 사람 얼굴이나 손이 이상하게 나옴**
- ❌ 얼굴·손이 포함된 사실적 이미지 요청 시 왜곡 발생
- ✅ 재생성(Re-generate)을 여러 번 시도하거나 구도를 바꿔 요청
- 💡 **해결법**: "얼굴은 측면으로", "손은 화면 밖으로" 등 구도 조정

**3. 저작권 있는 스타일/캐릭터 요청**
- ❌ "피카소 스타일로", "미키마우스를 그려줘"
- ✅ "큐비즘 스타일로", "귀여운 마우스 캐릭터" 등 일반적 표현 사용
- 💡 **해결법**: 특정 예술가나 캐릭터 이름 대신 스타일 특징을 묘사하세요
