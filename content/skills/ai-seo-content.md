## 🎯 학습 목표

1. AI로 검색 의도(Search Intent)를 분석하고 콘텐츠 방향을 설정할 수 있다
2. SEO에 최적화된 글 구조(제목, 소제목, 메타 설명)를 AI로 생성할 수 있다
3. 키워드 리서치→구조→작성→최적화 전체 파이프라인을 AI와 함께 완성할 수 있다

---

## 📖 AI SEO 콘텐츠의 핵심 원칙

구글은 EEAT(경험·전문성·권위성·신뢰성)를 기반으로 콘텐츠를 평가합니다. AI가 초안을 쓰더라도 반드시 실제 경험과 전문성을 담아야 합니다.

**AI SEO 콘텐츠 4단계:**
1. **키워드 리서치** — 타겟 키워드와 관련 키워드 발굴
2. **구조 설계** — 검색 의도에 맞는 목차 구성
3. **초안 작성** — AI로 본문 생성
4. **SEO 최적화** — 메타태그, 내부링크, E-E-A-T 보강

---

## 🛠️ 실습 1 — 키워드 리서치 및 콘텐츠 기획

```
SEO 콘텐츠 기획을 도와주세요.

내 블로그/사이트: [주제 또는 URL]
목표 독자: [타겟 독자층]
작성하려는 주제: [주제]

다음을 제공해주세요:
1. 메인 키워드 1개 (월 검색량 많고 경쟁 낮은 것)
2. LSI(의미 연관) 키워드 10개
3. People Also Ask 스타일 질문 5개
4. 이 키워드로 검색하는 사람의 검색 의도 분석
5. 경쟁 콘텐츠를 이길 수 있는 차별화 포인트 제안
```

## 🛠️ 실습 2 — SEO 최적화 글 작성 파이프라인

**1단계: 구조 설계**
```
키워드 "[메인 키워드]"로 SEO 블로그 포스트 구조를 만들어주세요.

검색 의도: [정보형/탐색형/거래형]
타겟 독자: [독자층]
희망 분량: [1500/2000/3000자]

결과물:
- SEO 최적화 제목 (H1) 3가지 후보
- 목차 구조 (H2 5개, 각 H2 아래 H3 2개)
- 각 섹션에 포함할 핵심 내용 한 줄씩
```

**2단계: 섹션별 작성**
```
위 구조의 "[섹션명]" 섹션을 작성해주세요.

포함 사항:
- 메인 키워드 "[키워드]" 자연스럽게 2~3회 포함
- LSI 키워드: [키워드 목록]
- 전문성을 보여주는 구체적 예시 또는 데이터
- 독자가 바로 실행할 수 있는 팁
분량: [글자수]
```

## 💬 프롬프트 템플릿 8개

### 1. 메타 디스크립션 생성
```
아래 블로그 포스트의 SEO 메타 디스크립션을 작성해주세요.

포스트 제목: [제목]
메인 키워드: [키워드]
포스트 핵심 내용: [요약]

조건:
- 150~160자 이내
- 메인 키워드 포함
- 클릭을 유도하는 CTA 포함
- 자연스러운 한국어

3가지 버전으로 작성해주세요.
```

### 2. FAQ 섹션 생성
```
"[주제]" 관련 FAQ 섹션을 작성해주세요.

질문 5개를 만들어주세요.
조건:
- 실제로 사람들이 검색할 법한 질문
- 각 답변: 2~3문장으로 간결하게
- 구조화된 데이터(Schema)에 적합한 Q&A 형식
- 키워드: [키워드들] 자연스럽게 포함
```

### 3. 내부 링크 전략
```
아래 블로그 포스트에 삽입할 내부 링크 전략을 세워주세요.

현재 포스트 주제: [주제]
기존 포스트 목록: [포스트 제목들]

제안 사항:
- 어떤 포스트로 링크할지 (앵커 텍스트 포함)
- 링크 삽입 위치 (어느 단락에)
- 링크 방향 (현재→기존, 기존→현재)
```

### 4. 경쟁 콘텐츠 분석
```
"[키워드]"로 검색되는 상위 콘텐츠들을 이길 수 있는
차별화 전략을 제안해주세요.

내 포스트의 강점: [특징]
일반적인 경쟁 콘텐츠의 약점을 추측해서:
1. 내 콘텐츠가 제공해야 할 고유 가치
2. 추가해야 할 섹션/내용
3. 포함하면 차별화되는 요소 (사례/데이터/인터뷰 등)
```

### 5. 제목 A/B 테스트 후보
```
아래 블로그 포스트의 SEO 제목 후보를 10개 만들어주세요.

주제: [주제]
메인 키워드: [키워드]

각 유형별로 2개씩:
- 숫자 포함형 ("7가지", "10분 만에")
- 질문형 ("~하는 법은?")
- 혜택 강조형 ("~로 X 달성하기")
- 문제 해결형 ("~를 해결하는 방법")
- 권위 강조형 ("전문가가 알려주는 ~")
```

### 6. 이미지 ALT 텍스트 최적화
```
아래 블로그 포스트에 사용할 이미지들의 ALT 텍스트를 작성해주세요.

이미지 설명:
1. [이미지1 설명]
2. [이미지2 설명]

조건:
- 각 ALT 텍스트 125자 이내
- 키워드 "[키워드]" 자연스럽게 포함
- 이미지 내용을 정확히 설명
- 과도한 키워드 반복 없이
```

### 7. 업데이트된 기존 포스트 최적화
```
아래 기존 블로그 포스트를 SEO 업데이트해주세요.

포스트 내용: [내용 붙여넣기]
업데이트 목적: [더 높은 순위 / 최신 정보 반영]

개선 사항:
1. 제목 최적화 제안
2. 추가해야 할 키워드 섹션
3. 삭제하거나 줄여야 할 내용
4. 최신화가 필요한 정보
5. 내부/외부 링크 추가 제안
```

### 8. 롱테일 키워드 콘텐츠 클러스터
```
"[메인 키워드]" 주제로 콘텐츠 클러스터를 설계해주세요.

필러 페이지(메인): 1개
클러스터 페이지(서브): 5개

각 페이지마다:
- 타겟 키워드
- 예상 월 검색량 (추정)
- 콘텐츠 형식 (방법/가이드/비교/리뷰 등)
- 메인 페이지와의 연결 방법
```

## 🎯 도전 과제

**미션**: 하나의 키워드로 SEO 최적화 블로그 포스트를 완성하고 실제 발행하세요.

**완성 조건:**
- 키워드 리서치 → 구조 → 작성 → 최적화 전 단계 완료
- 1500자 이상, 메타 디스크립션·ALT 텍스트 포함
- 실제 블로그(티스토리/네이버블로그/WordPress 등)에 발행
- Google Search Console에 URL 제출

## ⚠️ 자주 하는 실수 3가지

**1. AI가 쓴 글을 수정 없이 그대로 발행**
- ❌ AI 초안 → 즉시 발행
- ✅ 개인 경험·데이터·의견을 추가해 EEAT 강화
- 💡 **해결법**: AI 글의 최소 30%는 나만의 경험·인사이트로 대체

**2. 키워드를 억지로 많이 넣음 (키워드 스터핑)**
- ❌ 동일 키워드를 문단마다 반복
- ✅ 자연스러운 흐름 유지, LSI 키워드로 다양화
- 💡 **해결법**: 키워드 밀도보다 독자가 읽기 좋은가를 우선 기준으로

**3. 검색 의도를 무시한 콘텐츠**
- ❌ 키워드는 맞는데 독자가 원하는 내용이 아님
- ✅ 해당 키워드로 검색하는 사람이 진짜 원하는 것 분석 후 작성
- 💡 **해결법**: AI에게 "이 키워드로 검색하는 사람의 진짜 의도는?"을 먼저 물어보세요
