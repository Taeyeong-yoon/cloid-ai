## 키보드 단축키

| 단축키 | 기능 |
|--------|------|
| `Enter` | 메시지 전송 |
| `Shift + Enter` | 줄바꿈 (전송 안 됨) |
| `Ctrl + K` | 새 대화 시작 |
| `Ctrl + /` | 단축키 목록 보기 |
| `Esc` | 응답 생성 중단 |
| `Ctrl + Shift + C` | 마지막 응답 복사 |

## 잘 모르는 기능들

### 응답 재생성
Claude 답변 하단의 **재생성** 아이콘(🔄)을 클릭하면 같은 질문으로 다른 버전의 답변을 받을 수 있습니다.

### 특정 부분만 편집
내 메시지 위에 마우스를 올리면 **연필 아이콘**이 나타납니다. 클릭해서 내용을 수정하면 그 시점부터 대화를 다시 시작합니다.

### 응답 길이 조절 프롬프트
- 짧게: "한 문단으로만"
- 길게: "모든 케이스를 빠짐없이 설명해서"
- 구조적으로: "목차 형식으로", "표로 정리해서"

### 대화 공유
대화 우측 상단 **Share** → 링크 생성 → 누구나 읽기 전용으로 접근 가능

### 음성 모드
모바일 앱에서 마이크 아이콘 클릭 → Claude와 음성으로 대화. 운전 중, 손이 바쁠 때 유용합니다.

## 프롬프트 패턴 단축키 (직접 만들기)

Project Instructions에 자주 쓰는 패턴을 저장해두고 짧은 키워드로 호출:

```
[프로젝트 지시사항]
사용자가 "/요약"이라고 하면: 대화 전체 내용을 3줄로 요약해줘
사용자가 "/표로"라고 하면: 이전 내용을 표 형식으로 재구성해줘
사용자가 "/번역"이라고 하면: 이전 내용을 영어로 번역해줘
```

## 모바일 앱 팁

- 롱프레스(꾹 누르기) → 메시지 복사·공유
- 스와이프 → 대화 삭제
- 위젯 추가 → 홈화면에서 바로 Claude 시작
