## 🛠️ 실습 1 — ChatGPT로 CSV 파일 분석

ChatGPT에 CSV 파일을 업로드하고 아래 프롬프트를 사용하세요.

```
첨부한 CSV 파일을 분석해주세요.

다음 순서로 진행해주세요:
1. 데이터 구조 요약 (행/열 수, 컬럼 목록, 데이터 타입)
2. 기초 통계 (평균, 최소, 최대, 중앙값)
3. 결측치(빈 값) 및 이상치 확인
4. 가장 흥미로운 패턴이나 인사이트 3가지
5. 추가로 분석하면 좋을 것 같은 질문 3가지 제안
```

## 🛠️ 실습 2 — AI 생성 Python 코드로 분석

Claude에게 분석 코드를 요청하고 Google Colab에서 실행하세요.

```
아래 데이터 구조의 CSV 파일을 분석하는 Python 코드를 작성해주세요.

파일명: sales_data.csv
컬럼 구성:
- date: 날짜 (YYYY-MM-DD)
- product: 제품명
- region: 지역 (서울/부산/대구/인천)
- sales_amount: 매출액 (원)
- quantity: 판매 수량

필요한 분석:
1. 월별 매출 추이 (라인 그래프)
2. 지역별 매출 비교 (막대 그래프)
3. 제품별 판매 수량 상위 10개 (가로 막대 그래프)

라이브러리: pandas, matplotlib, seaborn
코드에 한국어 주석 포함
Google Colab에서 실행 가능하게 작성
```

## 💬 프롬프트 템플릿 8개

### 1. 데이터 기초 탐색
```
첨부한 데이터를 탐색해주세요.

확인 사항:
- 전체 행/열 수
- 각 컬럼의 데이터 타입
- 결측치(빈 값) 비율
- 수치형 컬럼의 기초 통계 (mean, std, min, max)
- 카테고리형 컬럼의 고유값 수

분석 결과를 표 형태로 정리해주세요.
```

### 2. 비즈니스 인사이트 추출
```
아래 매출 데이터를 분석해서 비즈니스 인사이트를 추출해주세요.

[데이터 붙여넣기 또는 파일 첨부]

알고 싶은 것:
1. 매출이 가장 높은 시간대/요일/월은?
2. 매출 성장률이 가장 높은 제품 카테고리는?
3. 개선이 필요한 영역은 어디인가?
4. 다음 분기 매출 예측 (단순 추세 기반)

의사결정에 바로 활용할 수 있는 형태로 요약해주세요.
```

### 3. 두 데이터셋 비교
```
두 기간의 데이터를 비교 분석해주세요.

전기 데이터: [내용 또는 파일]
후기 데이터: [내용 또는 파일]

비교 기준:
- 주요 지표 변화율 (%)
- 가장 크게 변한 항목 Top 5
- 전기 대비 개선된 점, 악화된 점
- 변화 원인 가설 제안
```

### 4. 이상치 탐지
```
아래 데이터에서 이상치(outlier)를 찾아주세요.

[데이터 붙여넣기]

요청 사항:
- 통계적 방법으로 이상치 식별 (IQR 또는 Z-score)
- 이상치 행 목록과 해당 값
- 이상치가 분석에 미치는 영향
- 이상치 처리 방법 제안 (제거/대체/유지)
```

### 5. 상관관계 분석
```
아래 변수들 사이의 상관관계를 분석해주세요.

변수 목록: [컬럼 이름들]
[데이터 붙여넣기]

분석 내용:
- 각 변수 쌍의 상관계수 (피어슨/스피어만)
- 상관관계가 가장 강한 조합 3가지
- 비즈니스 관점에서 의미 있는 인사이트
- 상관관계 히트맵 코드 (Python)
```

### 6. 고객 세분화 분석
```
아래 고객 데이터로 고객 세그먼트를 분석해주세요.

[고객 데이터 붙여넣기]
(구매 빈도, 최근 구매일, 구매 금액 포함)

분석 방법: RFM 분석 (Recency, Frequency, Monetary)
결과물:
- 각 고객의 RFM 점수
- 세그먼트별 고객 수와 특징
- 각 세그먼트별 마케팅 전략 제안
```

### 7. SQL 쿼리 생성
```
아래 데이터베이스 스키마에서 원하는 데이터를 추출하는
SQL 쿼리를 작성해주세요.

테이블 구조:
orders (order_id, customer_id, product_id, amount, order_date)
customers (customer_id, name, region, signup_date)
products (product_id, name, category, price)

원하는 결과: [원하는 데이터 설명]
데이터베이스: MySQL
쿼리와 함께 동작 원리 설명 포함
```

### 8. 자동화 리포트 생성
```
아래 데이터로 주간 리포트를 자동 생성하는 Python 코드를 만들어주세요.

데이터 구조: [설명]
리포트 내용:
- 핵심 지표 요약 (KPI)
- 전주 대비 변화
- 시각화 차트 3개
- 주요 인사이트 텍스트

출력: HTML 리포트 파일로 저장
```

## 🎯 도전 과제

**미션**: 실제 업무 데이터(또는 공개 데이터)를 AI로 분석하고 인사이트 보고서를 작성하세요.

**완성 조건:**
- 100행 이상의 실제 데이터 사용
- AI와 대화하며 최소 5가지 질문으로 분석
- 인사이트 3가지 + 실행 제안 2가지 포함한 보고서 완성

---

## ⚠️ 자주 하는 실수 3가지

**1. 데이터 설명 없이 바로 분석 요청**
- ❌ 파일만 업로드하고 "분석해줘"
- ✅ 데이터의 배경(언제 수집, 무엇을 측정, 비즈니스 컨텍스트)을 함께 설명
- 💡 **해결법**: "이 데이터는 ~를 기록한 것입니다. ~를 알고 싶습니다"로 시작

**2. AI 분석 결과를 검증 없이 보고서에 사용**
- ❌ AI가 계산한 수치를 그대로 최종 보고서에 포함
- ✅ 핵심 수치는 엑셀 또는 코드로 직접 검증
- 💡 **해결법**: 중요한 수치는 항상 교차 검증, AI는 방향성 탐색에 활용

**3. 분석 목적 없이 탐색만 하다 끝남**
- ❌ "재미있는 게 있나 봐줘" 식의 열린 탐색
- ✅ "이 데이터로 [특정 의사결정]을 하려고 한다"는 목적 먼저 설정
- 💡 **해결법**: 분석 전 "이 분석으로 무엇을 결정할 것인가?"를 먼저 답하세요
