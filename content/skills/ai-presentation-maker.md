## 🎯 학습 목표

1. 발표 목적과 청중에 맞는 스토리 구조를 AI로 설계할 수 있다
2. 각 슬라이드의 텍스트·메시지를 AI 프롬프트 체인으로 생성할 수 있다
3. AI가 생성한 내용을 실제 PPT/Canva에 빠르게 옮길 수 있다

---

## 📖 AI 발표자료 제작의 3단계

발표자료는 3단계 프롬프트 체인으로 만들 수 있습니다.

1. **기획**: 목적·청중·핵심 메시지 설정
2. **구성**: 슬라이드 목차와 각 슬라이드 내용 생성
3. **다듬기**: 제목·불릿포인트·스피커 노트 최적화

---

## 🛠️ 실습 1 — 기획 단계: 스토리 구조 잡기

```
발표자료 기획을 도와주세요.

발표 정보:
- 주제: [발표 주제]
- 청중: [누구에게 발표하는지]
- 목적: [정보 전달/설득/보고/판매]
- 시간: [발표 시간] (예: 10분)
- 슬라이드 수: [희망 슬라이드 수]

위 정보를 바탕으로:
1. 핵심 메시지 1문장
2. 3막 구조 (도입-전개-결론) 스토리라인
3. 각 섹션에 들어갈 슬라이드 목차
를 작성해주세요.
```

---

## 🛠️ 실습 2 — 구성 단계: 슬라이드 내용 생성

위 기획 결과를 바탕으로 각 슬라이드 내용을 생성하세요.

```
위에서 만든 목차를 바탕으로 각 슬라이드 내용을 작성해주세요.

각 슬라이드마다:
- 슬라이드 제목 (10자 이내, 임팩트 있게)
- 핵심 불릿포인트 3개 (각 15자 이내)
- 스피커 노트 (발표자가 말할 내용, 3~4문장)
- 추천 시각 요소 (그래프/사진/아이콘 등 제안)

총 [N]개 슬라이드 전부 작성해주세요.
```

## 💬 프롬프트 템플릿 5개

### 1. 비즈니스 제안 발표
```
다음 조건으로 비즈니스 제안 발표 자료를 만들어주세요.

제안 내용: [무엇을 제안하는지]
청중: [임원/팀장/클라이언트]
핵심 설득 포인트: [왜 이 제안이 좋은지 3가지]
슬라이드 수: 10장
구조: 문제 제기 → 해결책 → 기대 효과 → 실행 계획 → CTA
```

### 2. 업무 보고 발표
```
월간/주간 업무 보고 발표 자료를 만들어주세요.

보고 내용:
- 이번 기간 주요 성과: [내용]
- 진행 중인 프로젝트: [내용]
- 문제점 및 해결 방안: [내용]
- 다음 기간 계획: [내용]

형식: 5장 이내, 데이터/숫자 강조, 간결하게
```

### 3. 교육/강의 발표
```
[주제]에 관한 교육 발표 자료를 만들어주세요.

대상: [수강생 특성]
시간: [분]
학습 목표: [수강생이 얻어야 할 것]
구조: 오리엔테이션 → 핵심 개념 3가지 → 실습/예시 → Q&A 준비
각 슬라이드에 "강사 팁" 섹션 포함
```

### 4. 슬라이드 제목 개선
```
아래 발표 슬라이드 제목들이 너무 평범합니다.
더 임팩트 있고 청중의 관심을 끄는 제목으로 바꿔주세요.

원래 제목들:
1. [제목1]
2. [제목2]
3. [제목3]

조건: 10자 이내, 질문형/숫자 포함/감정 자극 형태로
```

### 5. 스피커 노트 생성
```
아래 슬라이드 내용을 바탕으로 발표자 노트를 작성해주세요.

슬라이드 제목: [제목]
슬라이드 내용: [불릿포인트 목록]

스피커 노트 요구사항:
- 자연스러운 말하기 스타일로
- 예시나 짧은 스토리 1개 포함
- 다음 슬라이드로 넘어가는 연결 문장 포함
- 읽는 데 약 [N]분 분량
```

## 🎯 도전 과제

**미션**: 실제 발표가 있거나 연습용 주제로 10장짜리 발표자료를 AI로 완성하세요.

**완성 조건:**
- 기획(1단계) → 구성(2단계) → 슬라이드 작성(3단계) 프로세스 완료
- Canva, PowerPoint, 또는 Google Slides에 실제 슬라이드 10장 완성
- 발표 리허설 1회 진행 (혼자라도 OK)

---

## ⚠️ 자주 하는 실수 3가지

**1. 청중을 고려하지 않은 발표 구성**
- ❌ 모든 정보를 다 넣으려 함
- ✅ "이 발표를 보는 [청중]에게 가장 중요한 메시지 하나는 무엇인가?"를 먼저 정의
- 💡 **해결법**: AI에게 "이 청중 입장에서 가장 궁금할 질문 3가지를 먼저 알려줘"로 시작

**2. 슬라이드에 텍스트를 너무 많이 넣음**
- ❌ AI가 생성한 긴 문장을 슬라이드에 그대로 붙여넣음
- ✅ "각 불릿포인트를 10자 이내로 줄여줘"로 재요청
- 💡 **해결법**: 슬라이드는 키워드만, 세부 내용은 스피커 노트에

**3. 디자인 단계를 AI가 다 해줄 것으로 기대**
- ❌ AI가 예쁜 슬라이드를 직접 만들어줄 거라고 기대
- ✅ AI는 내용(텍스트)을 생성, 디자인은 Canva/PowerPoint 템플릿 활용
- 💡 **해결법**: Canva에서 템플릿 선택 후 AI가 만든 텍스트를 붙여넣는 방식 추천
