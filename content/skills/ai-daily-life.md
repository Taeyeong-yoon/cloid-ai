## 🎯 학습 목표

1. 반복되는 이메일·문서 작업을 AI 프롬프트로 자동화할 수 있다
2. 흩어진 메모와 회의 내용을 AI로 구조화할 수 있다
3. 일상에서 매일 쓰는 나만의 프롬프트 템플릿을 3개 이상 만들 수 있다

---

## 📖 왜 AI 자동화인가?

직장인 평균 하루 업무 시간의 28%가 이메일 처리에 쓰입니다. AI를 활용하면 이 시간을 70% 이상 줄일 수 있습니다. 코딩 없이, 앱 설치 없이, ChatGPT·Claude 대화창 하나로 충분합니다.

## 🛠️ 실습 1 — 이메일 자동화

아래 상황을 AI에게 주고 이메일을 받아보세요.

```
상황: 중요한 미팅을 갑자기 취소해야 합니다.
- 상대방: 협력사 김 부장님
- 이유: 내부 긴급 회의 발생
- 대안: 다음 주 중 재조율 요청
- 톤: 정중하고 간결하게, 죄송함 표현 포함

위 상황에 맞는 이메일을 작성해주세요.
제목도 함께 써주세요.
```

---

## 🛠️ 실습 2 — 메모 구조화

회의 중 대충 적은 메모를 AI에게 넘기세요.

```
아래는 오늘 팀 회의에서 내가 대충 적은 메모입니다.
이것을 보기 좋게 정리해주세요:
- 회의 요약 (3줄 이내)
- 결정 사항 목록
- 액션아이템 (담당자, 기한 포함)

[메모 붙여넣기]
```

## 💬 프롬프트 템플릿 10개

### 1. 정중한 거절 이메일
```
아래 상황에서 정중하게 거절하는 이메일을 써주세요.
상황: [상황 설명]
관계: [상대방과의 관계]
대안 제시 여부: [있음/없음]
길이: 4문장 이내
```

### 2. 일정 조율 이메일
```
미팅 일정을 잡는 이메일을 써주세요.
- 목적: [미팅 목적]
- 후보 날짜: [3개 제시]
- 예상 소요 시간: [시간]
- 장소/방식: [온라인/오프라인]
```

### 3. 업무 보고 이메일
```
상사에게 보내는 주간 업무 보고 이메일을 써주세요.
완료한 일: [목록]
진행 중인 일: [목록]
다음 주 계획: [목록]
문제점/요청사항: [내용]
```

### 4. 감사 이메일
```
[상황]에 대한 감사 이메일을 써주세요.
진심이 느껴지되 과하지 않게, 3문장 이내로 작성해주세요.
```

### 5. 오늘 할 일 우선순위 정리
```
오늘 해야 할 일 목록입니다:
[할 일 나열]

이것을 긴급도·중요도 기준으로 순위를 매기고,
각 작업의 예상 소요 시간을 알려주세요.
```

### 6. 회의 아젠다 작성
```
다음 회의의 아젠다를 만들어주세요.
회의 목적: [목적]
참석자: [명단]
시간: [분]
주요 논의 사항: [내용]
```

### 7. 복잡한 내용 요약
```
아래 긴 문서/이메일을 3줄로 요약해주세요.
핵심 포인트와 내가 취해야 할 행동이 있다면 알려주세요.

[내용 붙여넣기]
```

### 8. SNS 캡션 작성
```
아래 상황/사진을 설명하는 인스타그램 캡션을 써주세요.
분위기: [밝고 친근한 / 전문적인 / 감성적인]
해시태그: 5개 포함
길이: 3문장 이내
상황: [설명]
```

### 9. 일일 계획표 생성
```
오늘 나의 상황:
- 근무 시간: [시작~종료]
- 꼭 해야 할 일: [목록]
- 미팅: [시간대]
- 점심 시간: [시간]

이를 바탕으로 시간 블록 형태의 오늘 계획표를 만들어주세요.
```

### 10. 독서/영상 메모 정리
```
아래는 내가 [책/영상]을 보면서 적은 메모입니다.
이것을 다음 형식으로 정리해주세요:
1. 핵심 인사이트 3가지
2. 바로 적용할 수 있는 실천 사항 2가지
3. 기억할 명언/문구 1개

[메모 붙여넣기]
```

## 🎯 도전 과제

**미션**: 오늘 하루 실제 업무에서 AI를 최소 3번 활용하세요.

**완성 조건:**
- 이메일 1개 AI로 작성 완료
- 메모 또는 회의 내용 AI로 정리 완료
- 내일 할 일 목록을 AI로 우선순위 정렬 완료
- 위 3가지 결과물 스크린샷 저장

---

## ⚠️ 자주 하는 실수 3가지

**1. 컨텍스트 없이 요청하기**
- ❌ "이메일 써줘"
- ✅ 받는 사람, 목적, 톤, 길이를 모두 명시
- 💡 **해결법**: 5W1H(누가·언제·어디서·무엇을·왜·어떻게)를 포함하면 훨씬 좋은 결과가 나옵니다

**2. AI 결과를 무편집으로 사용**
- ❌ AI가 쓴 이메일을 그대로 발송
- ✅ 반드시 내 말투·상황에 맞게 한두 곳 수정 후 사용
- 💡 **해결법**: AI는 초안 생성 도구입니다. 최종 검토는 항상 내가 합니다

**3. 매번 같은 내용을 다시 입력**
- ❌ 매일 비슷한 이메일 상황을 처음부터 설명
- ✅ 자주 쓰는 프롬프트를 메모장에 저장해두고 붙여넣기
- 💡 **해결법**: 나만의 프롬프트 모음집을 만들어두세요
