## 무엇인가?

Cowork는 여러 사람이 하나의 Claude 대화에 참여해 함께 작업하는 기능입니다. 2025년 출시 후 Microsoft Wave 3 업데이트에서 Copilot과도 통합되었습니다.

## 주요 기능

- **실시간 공동 편집**: 팀원이 같은 대화에 메시지를 보내면 모두가 실시간으로 확인
- **공유 컨텍스트**: 팀 전체가 같은 맥락에서 Claude에게 질문
- **역할 분리**: 각 팀원이 다른 관점으로 질문하면 Claude가 종합적으로 답변
- **히스토리 공유**: 대화 기록을 팀 전체가 검색·참조 가능

## 사용 시나리오

### 브레인스토밍 세션
```
[팀장] 새 제품 이름 후보 10개 제안해줘
[디자이너] 그 중에서 시각적으로 표현하기 좋은 건 어떤 게 있어?
[마케터] 2030 세대에게 어필할 수 있는 이름으로 좁혀줘
```

### 코드 리뷰
```
[개발자A] 이 PR 코드 리뷰해줘 [코드 첨부]
[개발자B] 보안 취약점 관점에서도 봐줘
[팀장] 우선순위별로 수정 사항 정리해줘
```

### 문서 작성
```
[기획자] 기능 명세서 초안 작성해줘
[개발자] 기술적으로 불가능한 부분 있으면 표시해줘
[QA] 테스트 케이스 추가해줘
```

## 접근 방법

- claude.ai → 대화창 우측 상단 **Share / Cowork** 아이콘
- Claude Team/Enterprise 플랜에서 사용 가능
- Microsoft Copilot 사용자는 Copilot Chat에서 Claude 모드 전환 후 Cowork 기능 사용

## 팁

- Cowork 전에 Project Instructions로 팀 공통 규칙을 미리 설정하면 일관된 결과
- 중요한 결과물은 Artifact로 저장해서 팀 전체가 공유
- 역할을 명확하게 나누면 Claude의 답변 품질이 올라갑니다
