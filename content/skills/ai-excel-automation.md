## 🎯 학습 목표

1. 엑셀 수식 작성을 AI에게 요청하는 프롬프트 패턴을 익힌다
2. VLOOKUP, IF, SUMIF 등 자주 쓰는 수식을 AI로 생성할 수 있다
3. 오류가 난 수식을 AI에게 디버깅 요청할 수 있다

---

## 📖 왜 AI로 엑셀 수식을?

엑셀 함수는 400개 이상 존재하고, 복잡한 수식은 전문가도 검색해가며 작성합니다. AI에게 상황을 설명하면 즉시 완성된 수식을 받을 수 있습니다.

**AI 엑셀 활용의 3가지 장점:**
- 수식을 몰라도 원하는 결과를 얻을 수 있음
- 오류 원인을 즉시 파악하고 해결
- 매크로(VBA) 코드도 자동 생성

---

## 🛠️ 실습 1 — 수식 자동 생성

아래 프롬프트를 ChatGPT 또는 Claude에 입력해보세요.

```
엑셀 수식을 만들어주세요.

상황:
- A열: 직원 이름
- B열: 부서명
- C열: 판매 금액

목표:
- "영업팀" 직원들의 판매 금액 합계를 구하고 싶어요
- 결과를 E1 셀에 넣을 예정

어떤 수식을 써야 하나요? 수식과 함께 각 인자 설명도 해주세요.
```

---

## 🛠️ 실습 2 — 오류 수식 디버깅

수식이 오류날 때 AI에게 진단을 요청하세요.

```
아래 엑셀 수식에서 #VALUE! 오류가 납니다.
원인을 찾고 올바른 수식으로 고쳐주세요.

수식: =VLOOKUP(A2,Sheet2!B:D,2,FALSE)
상황: A2에는 직원 코드(숫자), Sheet2의 B열에도 직원 코드가 있는데 오류가 납니다.
```

## 💬 프롬프트 템플릿 — 업무 시나리오 5개

### 시나리오 1. 조건부 합계 (SUMIF/SUMIFS)
```
엑셀 SUMIFS 수식을 만들어주세요.

데이터 구조:
- A열: 날짜
- B열: 담당자 이름
- C열: 제품 카테고리
- D열: 매출액

원하는 결과: 2025년 1월에 "홍길동" 담당자의 "전자제품" 카테고리 매출 합계

수식과 각 인자 설명을 부탁해요.
```

### 시나리오 2. 데이터 찾기 (VLOOKUP/XLOOKUP)
```
엑셀 수식을 만들어주세요.

상황:
- Sheet1의 A열에 직원 ID가 있어요
- Sheet2의 A열에 직원 ID, B열에 이름, C열에 연봉이 있어요

목표: Sheet1의 A열 직원 ID를 기준으로
      Sheet2에서 해당 직원의 이름과 연봉을 가져오고 싶어요

VLOOKUP과 XLOOKUP 두 가지 방법 모두 알려주세요.
```

### 시나리오 3. 조건부 표시 (IF/IFS)
```
엑셀 IF 수식을 만들어주세요.

상황:
- B열에 점수가 있어요 (0~100)
- 90 이상: "A등급"
- 80 이상: "B등급"
- 70 이상: "C등급"
- 70 미만: "D등급"

이 조건으로 C열에 등급을 자동으로 표시하는 수식을 만들어주세요.
```

### 시나리오 4. 날짜 계산
```
엑셀 날짜 수식을 만들어주세요.

필요한 수식들:
1. 오늘 날짜 기준으로 계약 만료일(B열)까지 남은 날수
2. 입사일(C열)로부터 근속 연수 계산 (소수점 없이 정수로)
3. 이번 달 마지막 날짜 구하기

각각의 수식과 사용 방법을 알려주세요.
```

### 시나리오 5. VBA 매크로 생성
```
엑셀 VBA 매크로 코드를 만들어주세요.

원하는 기능:
- A열의 데이터가 있는 행들을 모두 읽어서
- B열 값이 "완료"인 행만 골라
- 새로운 시트("완료목록")에 복사하기

코드와 함께 실행 방법(어디에 붙여넣고 어떻게 실행하는지)도 설명해주세요.
```

## 🎯 도전 과제

**미션**: 현재 내가 수동으로 계산하는 엑셀 작업 하나를 AI 수식으로 자동화하세요.

**완성 조건:**
- 자동화할 작업 1개 선정
- AI에게 수식 요청 → 수식 적용 → 결과 확인 완료
- 이 수식으로 절약되는 시간(분/회)을 계산해보기

## ⚠️ 자주 하는 실수 3가지

**1. 데이터 구조 설명 없이 수식만 요청**
- ❌ "VLOOKUP 수식 만들어줘"
- ✅ 어떤 시트·어떤 열에 어떤 데이터가 있는지 구체적으로 설명
- 💡 **해결법**: 열 이름(A, B, C)과 각 열의 데이터 종류를 프롬프트에 포함

**2. AI 수식을 그대로 복붙하다 오류 발생**
- ❌ 수식 붙여넣기 후 오류가 나도 무엇이 문제인지 모름
- ✅ AI가 설명한 각 인자가 내 파일과 맞는지 확인 후 붙여넣기
- 💡 **해결법**: "이 수식에서 내 상황에 맞게 바꿔야 할 부분이 어디인가요?"로 추가 질문

**3. 오류 메시지를 AI에게 안 알려줌**
- ❌ #REF!, #VALUE! 등 오류가 났을 때 혼자 해결하려 시도
- ✅ 오류 메시지 + 수식 + 데이터 구조를 AI에게 설명하면 즉시 해결
- 💡 **해결법**: 스크린샷보다 텍스트로 "수식: ~, 오류: ~, 상황: ~" 형태로 전달
