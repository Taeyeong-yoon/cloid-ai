Radar
AI 트렌드의 최전선 — 매일 업데이트

Radar는 Claude Code 업데이트, 새 AI 모델 출시, 개발 도구 변화 등 실무에서 바로 써야 할 AI 뉴스를 매일 큐레이션합니다. 단순 뉴스 나열이 아니라, 읽은 후 Learning·Labs로 바로 연결됩니다.

Radar 활용 단계

1

홈에서 "오늘의 업데이트" 확인

홈 화면 상단에 오늘의 Radar 1개가 자동으로 표시됩니다. 매일 아침 확인하는 습관을 만드세요.

2

관심 주제 태그 클릭

각 Radar 카드의 태그(예: claude-api mcp)를 클릭하면 같은 주제의 모든 콘텐츠를 볼 수 있습니다.

3

다음 단계로 이동 → Learning

Radar 상세 페이지 하단의 "이 주제 더 배우기 →" 버튼으로 관련 Learning 콘텐츠로 이동하세요.

Skills
바로 쓸 수 있는 실무 레시피

Skills는 프롬프트 레시피, 코드 패턴, 실무 활용법을 모아놓은 공간입니다. 복사해서 바로 쓸 수 있도록 구성되어 있습니다.

Skills 사용법

1

레시피 찾기

검색창 또는 태그로 원하는 주제(예: "이메일 작성", "코드 리뷰")를 검색합니다.

2

프롬프트 복사

오른쪽 상단의 복사 버튼을 눌러 레시피를 클립보드에 복사합니다.

3

AI 도구에 붙여넣기 + 실습

ChatGPT, Claude 등에 붙여넣고 결과를 확인합니다. Labs 실습과 연결해 더 깊이 배우세요.

📋 레시피 예시 — 업무 이메일 자동화

역할: 당신은 전문 비즈니스 이메일 작성 전문가입니다. 다음 정보를 바탕으로 정중하고 명확한 이메일을 작성해주세요: - 수신자: [직책/이름] - 목적: [미팅 요청 / 피드백 요청 / 결과 보고 등] - 핵심 내용: [3줄 이내로 정리] - 톤: [격식체 / 반격식체] 이메일은 제목 포함 5문장 이내로 작성해주세요.

Labs
직접 실행하며 배우는 실습 공간

Labs는 설치 → 설정 → 실행 → 결과 확인까지 단계별로 안내하는 실습 공간입니다. AI가 생성한 HTML 코드를 바로 미리볼 수 있는 특별 도구도 포함되어 있습니다.

⚠️
실습 시작 전 반드시 확인! Labs를 시작하기 전에 항상 사전 체크리스트를 확인하세요. 환경 준비가 안 된 상태로 시작하면 중간에 막히기 쉽습니다.

⚡HTML 코드 미리보기 — 직접 해보세요!

HTML 미리보기 사용법

1

AI에서 HTML 코드 받기

ChatGPT, Claude 등에서 "HTML로 만들어줘"라고 요청하면 완성된 HTML 코드를 받을 수 있습니다.

2

코드 붙여넣기 → 자동 미리보기

왼쪽 텍스트 영역에 붙여넣으면 오른쪽에 즉시 미리보기가 나타납니다.

3

HTML 파일로 저장

결과가 마음에 들면 "HTML 저장" 버튼으로 내 컴퓨터에 .html 파일로 저장합니다.
