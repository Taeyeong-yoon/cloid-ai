섹션 A

무엇이 일어난 건가

2026년 3월, Microsoft는 M365 Copilot "Wave 3"를 발표하며 Anthropic의 Claude를 공식 통합했습니다. 이것이 왜 중요한지 단계별로 이해하세요.

📌 핵심 사실 3가지

1. Copilot Cowork 출시

Claude Cowork의 에이전트 기술을 Microsoft 365 안에 넣었습니다. Word, Excel, PowerPoint, Outlook, Teams를 넘나들며 멀티 스텝 작업을 자동 실행합니다. 단순 챗봇이 아니라 "일을 대신 해주는 에이전트"입니다.

2. Copilot Chat에서 Claude 선택 가능

Frontier 프로그램 사용자는 일반 Copilot Chat에서 GPT 대신 Claude를 선택할 수 있습니다. 관리자가 M365 관리 센터에서 토글로 켜면 됩니다.

3. Copilot Studio에서 Claude로 에이전트 구축

커스텀 에이전트를 만들 때 Claude Sonnet 4.5나 Claude Opus 4.1을 기본 모델로 선택 가능합니다. 관리자가 비활성화하면 자동으로 OpenAI로 전환됩니다.

🤔 왜 Microsoft가 경쟁사의 AI를 넣었나?

Microsoft의 답: "멀티모델 전략". 하나의 AI 모델이 모든 작업에서 최고일 수 없다는 판단입니다. Copilot이 작업 유형에 따라 가장 적합한 모델을 자동으로 선택합니다.

Microsoft 임원 Jared Spataro의 말: "적어도 60일마다 새로운 최강 모델이 나온다. 하나에 올인하는 것은 위험하다."

배경: Microsoft는 OpenAI에 135억 달러를 투자했지만, 동시에 Anthropic과 300억 달러 규모의 Azure 컴퓨팅 계약을 맺었습니다. 두 회사의 AI를 모두 제공하겠다는 전략입니다.

💡 정리: Copilot은 이제 "어떤 모델을 쓰느냐"가 아니라 "어떤 작업을 하느냐"가 중심입니다. 작업에 따라 OpenAI, Claude, 또는 다른 모델이 자동 배정됩니다.

🔑 핵심 개념: Work IQ

Work IQ는 Copilot Cowork이 Claude Cowork과 가장 크게 다른 점입니다. 사용자의 Outlook 이메일, Teams 대화, Calendar 일정, SharePoint 파일, 이전 작업 패턴 등 회사 내 모든 업무 데이터를 맥락으로 활용합니다.

Claude Cowork은 사용자가 허용한 로컬 폴더만 봅니다. Copilot Cowork은 M365 전체 데이터 그래프를 봅니다. 이 차이가 기업 사용자에게 중요한 이유입니다.

섹션 D

Claude가 Copilot 어디에 들어가 있나

Claude는 M365 Copilot의 세 곳에 통합되었습니다.

1️⃣ Copilot Cowork — 멀티 스텝 에이전트

가장 핵심. Claude의 에이전트 하네스(작업을 분해하고 실행하는 엔진)를 그대로 가져와서, M365 데이터 위에서 실행합니다. 장시간 실행, 멀티 앱 횡단, 계획-실행-검증 루프가 가능합니다.

상태: Research Preview. 2026년 3월 말 Frontier 프로그램으로 확대 예정.

2️⃣ Copilot Chat — 대화형 AI 선택

일반 Copilot Chat에서 GPT 대신 Claude를 선택할 수 있습니다. Fortune 500의 90%가 쓰는 같은 인터페이스에서 모델만 바꿀 수 있는 것입니다.

조건: 관리자가 M365 관리 센터에서 Anthropic 모델을 활성화해야 합니다.

3️⃣ Copilot Studio — 커스텀 에이전트 구축

조직 맞춤형 AI 에이전트를 만들 때 기본 모델로 Claude Sonnet 4.5 또는 Opus 4.1을 선택할 수 있습니다. 관리자가 Anthropic을 비활성화하면 자동으로 OpenAI로 전환됩니다.

상태: 이미 사용 가능 (2025년 9월 첫 도입 → 2026년 확대).

💡 세 가지 모두 관리자(IT)가 토글로 켜야 사용 가능합니다. 개인이 혼자 활성화할 수 없습니다. 회사에서 M365를 쓰고 있다면 IT 관리자에게 문의하세요.

📋 Copilot + Claude 통합 핵심 정리

[ Copilot + Claude 통합이란 ]
• Microsoft M365 Copilot에 Anthropic의 Claude AI가 공식 통합
• "멀티모델 전략": 작업에 따라 GPT와 Claude 중 최적 선택
• 2026년 3월 Wave 3로 발표

[ Claude가 들어간 곳 3가지 ]
1. Copilot Cowork — 멀티 스텝 에이전트 (Claude 에이전트 엔진)
2. Copilot Chat — 대화형 AI에서 Claude 선택 가능
3. Copilot Studio — 커스텀 에이전트 구축 시 Claude 모델 선택

[ Claude Cowork vs Copilot Cowork ]
• Claude Cowork = 로컬 파일, 개인/소규모, $20/월~
• Copilot Cowork = M365 전체 데이터, 기업용, $99/user/월~
• 같은 Claude 두뇌, 다른 접근 범위와 보안 체계

[ 핵심 개념 ]
• Work IQ = M365의 업무 맥락 인텔리전스 레이어
• 멀티모델 = 하나의 AI에 올인하지 않고 최적 선택
• E7 = M365 E5 + Copilot + Agent 365 번들 ($99/user/월)
