📋 6대 AI 한 줄 정리

🟠 Claude (Anthropic) — 코딩·글쓰기·분석 정밀도 최고. 안전 중심.
🟢 ChatGPT (OpenAI) — 가장 범용적. 이미지 생성, 음성, 플러그인.
🔵 Gemini (Google) — 1M 토큰. Google 연동. 비디오 분석 최강.
🔴 Grok (xAI) — 최저 API 가격. X 실시간 데이터. 2M 토큰.
🟣 DeepSeek (중국) — 오픈소스. 초저비용. 놀라운 가성비.
🔷 Perplexity — 실시간 검색 + 출처 인용. 리서치 특화.

📋 용도별 최적 AI 치트시트

코딩/개발 → Claude (Claude Code) 또는 Grok (가성비)
글쓰기/편집 → Claude (반복 정제에 강함)
범용 만능 → ChatGPT (플러그인, 이미지, 음성)
Google 생태계 → Gemini (Gmail, Drive, Docs 연동)
Microsoft 생태계 → Copilot (Claude+GPT 멀티모델)
리서치/팩트체크 → Perplexity (출처 인용)
실시간 뉴스/SNS → Grok (X 데이터 접근)
이미지 생성 → ChatGPT (DALL-E) 또는 Midjourney
비디오 분석 → Gemini (1시간+ 비디오)
초저비용 대량처리 → DeepSeek 또는 Grok API
최고 품질 추론 → Claude Opus 4.6
최대 컨텍스트 → Grok (2M) 또는 Gemini (1M)
